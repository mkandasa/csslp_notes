CSSLP: Functional Security Requirements

Translate Business Need to Security Requirement

Business Stakeholder says:
"We want customers to be able to upload profile pictures so the site feels more personal."

Select the correct Functional Security Requirement derived from this:

"The system shall allow users to upload any file type."
"The system shall validate that uploaded files are images (JPG/PNG) and do not contain executable code."
"The system shall scan the file for viruses every 30 days." (Non-Functional/Operational)
"The system shall display the image immediately."
